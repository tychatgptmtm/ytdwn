# Telegram YouTube Downloader Bot

Готовый Telegram-бот на Python для скачивания видео по ссылке с YouTube и YouTube Shorts.

## Что умеет

- принимает ссылку на YouTube
- принимает ссылку на YouTube Shorts
- скачивает видео через `yt-dlp`
- если файл небольшой — отправляет как обычное видео
- если файл крупнее — отправляет как файл
- если файл слишком большой — пишет сообщение об ошибке
- подходит для деплоя на Render

## Файлы проекта

- `bot.py` — основной код бота
- `requirements.txt` — зависимости Python
- `render.yaml` — конфиг для Render Blueprint
- `runtime.txt` — версия Python
- `.gitignore` — исключения для Git

## Что нужно перед запуском

1. Создай бота через `@BotFather`
2. Получи токен
3. Залей этот проект в новый GitHub-репозиторий

## Как залить на Render

1. Открой Render
2. Нажми `New`
3. Выбери `Blueprint`
4. Подключи GitHub-репозиторий с этим проектом
5. Нажми `Apply`
6. После создания сервиса открой его
7. Зайди в `Environment`
8. Добавь переменную:
   - `BOT_TOKEN` = токен твоего бота
9. Запусти деплой, если он не стартовал сам

## Как работает логика отправки

В коде стоят консервативные лимиты:

- до `45 MB` — бот шлёт как видео
- до `49 MB` — бот шлёт как документ
- больше — пишет, что файл слишком большой

Если захочешь, потом можно поднять или поменять эти числа в `bot.py`.

## Важно

- Бот скачивает видео в папку `downloads/`
- После отправки файл удаляется
- Если YouTube временно меняет ограничения, некоторые ссылки могут не скачиваться

## Локальный запуск

Установить зависимости:

```bash
pip install -r requirements.txt
```

Запустить:

```bash
export BOT_TOKEN="твой_токен"
python bot.py
```
